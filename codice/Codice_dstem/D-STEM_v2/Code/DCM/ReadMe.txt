This folder collects the demos of the paper "Finazzi F, Fassò A (2014) D-STEM: A Software for the Analysis and Mapping of Environmental Space-Time Variables. Journal of Statistical Software; 62: 1-29".

The MATLAB scripts related to the demos have been adapted in order to run with the latest version of D-STEM. Demo detailed in Section 7.2 of the paper has been removed since no longer supported by D-STEM.

If you wish to run the original demos detailed in the above paper, please download D-STEM from https://www.jstatsoft.org/article/view/v062i06. The zip file includes source code and demos.
